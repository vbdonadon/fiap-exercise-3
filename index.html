Acessibilidade em nosso site!

Estrutura do HTML

HTML semântico é o uso de marcação HTML para reforçar a semântica, ou significado, das informações em páginas da Web e aplicativos da Web, em vez de meramente definir sua apresentação ou aparência.

Leitores de Tela também são extremamente beneficiados por um HTML semântico, utilizando as TAGs do HTML corretamente, ajuda o leitor de tela a passar a informação mais clara para os usuários.

Teclas de Atalhos

As teclas de atalho ajudando na navegação do usuário para os conteúdos principais do site, como menu principal, conteúdo e rodapé

Alto contraste

Alto contraste é uma opção para pessoas com médio ou grande déficit visual. Deixa o fundo da página totalmente preto com as letras em branco, podendo ser associado à troca do tamanho das letras.

Aumentar/Diminuir a Fonte

Aumentar tamanho das letras, com uma única possibilidade de aumento, suficientemente grande para inúmeras pessoas com déficit visual limitado. Esta opção de acessibilidade é para todas as páginas do site, não sendo necessário voltar a aumentar o tamanho das letras a cada página visitada. Quando acionado, em seu lugar aparece o texto "Tamanho Original", que faz o retorno. Todas as vezes que se voltar ao site as letras estarão no tamanho escolhido na última visita;

Tamanho do Texto

Para aumentar ou diminuir a fonte no nosso site, utilize os atalhos Ctrl+ (para aumentar) e Ctrl- (para diminuir) no seu teclado.
